Matching

  • Built-in reference databases: FooDB (9,913 items) and DFG2 (256 items)

Review

  • Guided review workflow with keyboard-driven decisions
  • Bulk actions: select all, multi-select, range select, click-and-drag
  • Session persistence -- results and review decisions auto-save

Import & Export

  • CSV and TSV import/export with auto-detected delimiters
  • Original columns preserved, match metadata appended
